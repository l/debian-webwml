conffiles

Une des choses les plus énervantes avec les logiciels est quand vous avez passé un bon bout de temps à configurer un programme pour qu'une mise à jour efface tous vos changements. Debian résout ce problème en marquant les fichiers de configurations, de sorte que quand vous mettez à jour un programme, il vous sera demandé si vous voulez garder votre ancienne configuration ou pas. Vous faites ceci en entrant le chemin complet de chaque fichier de configuration (ils sont en général dans /etc), un par ligne, dans un fichier nommé conffiles. Empire n'a aucun fichier de configuration.
