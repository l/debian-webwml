Les Serveurs Internet

2.1 Les listes de diffusion

Le serveur des listes de diffusion est hébergé sur lists.debian.org. Envoyez un courrier électronique à debian-foo-REQUEST@lists.debian.org, debian-foo étant le nom de la liste, avec le mot subscribe dans le sujet pour vous abonner ou unsubscribe pour vous désabonner.

Quand vous répondez à un message sur la liste, n'envoyez pas de copie en CC à l'auteur. Toute personne écrivant à une liste de diffusion devrait la lire pour prendre connaissance des réponses.

Comme toujours dans vos courriers, essayez de réduire les citations des articles auxquels vous répondez. Appliquez vous aussi à respecter la Nétiquette lorsque vous postez.
