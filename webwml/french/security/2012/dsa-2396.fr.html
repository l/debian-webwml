Bulletin d'alerte Debian

DSA-2396-1 qemu-kvm -- buffer underflow

Date du rapport :
27 janvier 2012
Paquets concernés :
qemu-kvm
Vulnérabilité :
Oui
Références dans la base de données de sécurité :
Dans le dictionnaire CVE du Mitre : CVE-2012-0029.
Plus de précisions :

Nicolae Mogoraenu a découvert a heap overflow dans the emulated e1000e network interface card de KVM, a solution pour full virtualization on x86 hardware, which pourrait result dans déni de service ou augmentation de droits.

This update also fixes a guest-triggerable memory corruption dans VNC traitement.

Pour la distribution stable (Squeeze), ce problème a été corrigé dans la version 0.12.5+dfsg-5+squeeze8.

Pour la distribution unstable (Sid), ce problème a été corrigé dans la version 1.0+dfsg-5.

Nous vous recommandons de mettre à jour vos paquets qemu-kvm.
