Fecha de notificación:
20 de Febrero de 1997.
Paquetes afectados:
screen
Breve descripción:
El programa "screen" se desborda al copiar el campo gcos.
Vulnerable:
Existe el desbordamiento, pero screen renuncia a privilegios de root antes de que se ejecute la parte defectuosa del código.
