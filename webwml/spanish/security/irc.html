Información sobre Seguridad

Por favor, envie las noticias de fallos relacionados con la seguridad a security@debian.org

Fecha notificación: 8 de Mayo de 1998

Paquetes afectados: irc previo a las versiones listadas abajo

Breve descripción: irc permite potencialmente el envio de caractéres a terminal local
David Holland ha informdo de que un usuario IRC II remoto puede enviar caractéres arbitrarios - códigos ansi - a la terminal de un usuario. Esto se considera dañino. Las siguientes versiones corrigen este problema.

Vulnerable: Sí
