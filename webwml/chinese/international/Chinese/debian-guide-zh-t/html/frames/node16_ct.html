12 移除和安裝軟件

本章介紹安裝和移除軟件包的方法。Debian 提供幾種方式進行這些操作，我們在這裡討論安裝和移除編譯好的軟件（例如 Debian 軟件包）和安裝源碼文件（這些你必須自己編譯）。

1 軟件包維護工具做些什麼

一個應用軟件或者工具程序常常包含了不少的文件，可能的文件有程序庫，數據文件，像遊戲腳本或者圖標，配置文件，用戶手冊和說明書。在安裝的時候，你需要把所有的文件安裝在正確的地方。

你也會需要反安裝某些程序。當你反安裝的時候，你會想把所有相關的文件一起刪除。同時，如果你系統中的另外一個程序也需要這些文件的時候，那這些文件當然是需要保証不被刪除的。

最後，你也希望能夠都某個程序進行升級。升級的時候，要把舊的、作廢的文件刪除，加入新的文件，同時還不能破壞系統的任何部分。

Debian 的軟件包裝系統專門解決這些問題。它讓你安裝、移除、升級軟件包，軟件包裡有程序文件和其他資料，這些資料幫助電腦正確的管理軟件包。Debian 的軟件包的文件名後綴是 .deb，你可以從你的 Debian 正式光碟裡或者 FTP 站上找到這些軟件包。


2 dpkg（Debian 軟件包系統）

安裝一個你下載的軟件包的最簡單的方法，是使用命令dpkg -i（是dpkg -install 的縮寫）。假設你下載了一個軟件包 icewm_0.8.12-1.deb 並且要安裝它，首先，以 root 登錄，然後輸入dpkg -i icewm_0.8.12-1.deb，icewm 版本 0.8.12 就已經安裝好了。 如果你已經有一個比較舊的版本，dpkg 會把你的舊版本升級，而不是把兩個版本分別裝上。

如果你想移除一個軟件包，你有兩個選擇。第一個是最直觀的：dpkg -r icewm。這個命令會把 icewm 軟件包移除（-r 是 -remove 的縮寫）。注意這裡你只給 -remove 軟件包名字 icewm，而在 -install 時，需要使用帶 .deb 的完整的文件名。

-remove 文件，你可能通過修改這個文件而改變系統裡的程序或者你的習慣參數。使用這種方式，如果以後你又重新安裝了這個軟件包，你不用重新設置所有的設定。

但有時候你會想要把這些配置文件也一起抹去，所以 dpkg 也提供一個淨化選項 -purge（淨化，清除）。dpkg -purge icewm 會永久的刪除所有跟 icewm 軟件包有關的任何文件。


3 dselect (Debian select == Debian 選擇)

dselect 是一個很酷的 dpkg 前端（front-end, 簡單界面）。dselect 為 dpkg 提供一個選項界面，而且它可以自動的從光碟或者互聯網 FTP 上取得需要的文件。有關使用 dselect 的詳細資料，請參閱第 頁第 3.20 節。（在 dselect 裡按 ? 進入幫助系統，請務必熟悉 dselect 的使用方法。）

4 編譯軟件

許多程序是以源文件的形式發行的，一般是以 tar.gz 的形式包裝的。首先你要先把 tar.gz 文件解開﹔具體怎麼做參見第 頁第 10.3.1 節。編譯以前，你要安裝 gcc、libc6-dev，和其他一些有關``-dev'' 軟件包。這些包大多數是在 dselect 的 devel 區裡面的。

安裝好了這些相關的軟件包，cd（換目錄）到 tar 生成的目錄下面。仔細閱讀安裝指示，大多數程序提供一個 INSTALL 或 README 文件，這些文件會告訴你安裝步驟。
