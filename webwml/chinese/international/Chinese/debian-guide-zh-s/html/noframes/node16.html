12 移除和安装软件

本章介绍安装和移除软件包的方法。Debian 提供几种方式进行这些操作，我们在这里讨论安装和移除编译好的软件（例如 Debian 软件包）和安装源码文件（这些你必须自己编译）。

1 软件包维护工具做些什么

一个应用软件或者工具程序常常包含了不少的文件，可能的文件有程序库，数据文件，像游戏脚本或者图标，配置文件，用户手册和说明书。在安装的时候，你需要把所有的文件安装在正确的地方。

你也会需要反安装某些程序。当你反安装的时候，你会想把所有相关的文件一起删除。同时，如果你系统中的另外一个程序也需要这些文件的时候，那这些文件当然是需要保证不被删除的。

最后，你也希望能够都某个程序进行升级。升级的时候，要把旧的、作废的文件删除，加入新的文件，同时还不能破坏系统的任何部分。

Debian 的软件包装系统专门解决这些问题。它让你安装、移除、升级软件包，软件包里有程序文件和其他资料，这些资料帮助计算机正确的管理软件包。Debian 的软件包的文件名后缀是 .deb，你可以从你的 Debian 正式光盘里或者 FTP 站上找到这些软件包。


2 dpkg（Debian 软件包系统）

安装一个你下载的软件包的最简单的方法，是使用命令dpkg -i（是dpkg -install 的缩写）。假设你下载了一个软件包 icewm_0.8.12-1.deb 并且要安装它，首先，以 root 登录，然后输入dpkg -i icewm_0.8.12-1.deb，icewm 版本 0.8.12 就已经安装好了。 如果你已经有一个比较旧的版本，dpkg 会把你的旧版本升级，而不是把两个版本分别装上。

如果你想移除一个软件包，你有两个选择。第一个是最直观的：dpkg -r icewm。这个命令会把 icewm 软件包移除（-r 是 -remove 的缩写）。注意这里你只给 -remove 软件包名字 icewm，而在 -install 时，需要使用带 .deb 的完整的文件名。

-remove 文件，你可能通过修改这个文件而改变系统里的程序或者你的习惯参数。使用这种方式，如果以后你又重新安装了这个软件包，你不用重新设置所有的设定。

但有时候你会想要把这些配置文件也一起抹去，所以 dpkg 也提供一个净化选项 -purge（净化，清除）。dpkg -purge icewm 会永久的删除所有跟 icewm 软件包有关的任何文件。


3 dselect (Debian select == Debian 选择)

dselect 是一个很酷的 dpkg 前端（front-end, 简单界面）。dselect 为 dpkg 提供一个选项界面，而且它可以自动的从光盘或者互联网 FTP 上取得需要的文件。有关使用 dselect 的详细资料，请参阅第 页第 3.20 节。（在 dselect 里按 ? 进入帮助系统，请务必熟悉 dselect 的使用方法。）

4 编译软件

许多程序是以源文件的形式发行的，一般是以 tar.gz 的形式包装的。首先你要先把 tar.gz 文件解开；具体怎么做参见第 页第 10.3.1 节。编译以前，你要安装 gcc、libc6-dev，和其他一些有关``-dev'' 软件包。这些包大多数是在 dselect 的 devel 区里面的。

安装好了这些相关的软件包，cd（换目录）到 tar 生成的目录下面。仔细阅读安装指示，大多数程序提供一个 INSTALL 或 README 文件，这些文件会告诉你安装步骤。
