Security Information

Please send security-related bug reports to security@debian.org

Date reported: 08 May, 1998

Affected packages: irc prior to versions listed below

Brief description: irc potentially permits remote sending of characters to local terminal
David Holland has reported that a remote IRC II user may send arbitrary characters - ansi codes - to a users terminal. This is considered harmful. The following versions fix this problem.

Vulnerable: Yes
