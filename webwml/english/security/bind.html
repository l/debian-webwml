Security Information

Please send security-related bug reports to security@debian.org

Date reported: non-public - April 01, 1998; public - April 08, 1998

Affected packages: bind 4.9 prior to 4.9.7-1 and bind 8 prior to 8.1.2-1

Brief description:
Older versions of bind are vulnerable to a buffer overrun during an inverse query request, potentially allowing system access as root. Additional vulnerabilities permit denial of service attacks on the bind service.

Vulnerable: Yes
