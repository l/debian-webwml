Please send security-related bug reports to security@debian.org

Brief description:
GNU tar sometimes unintentionally creates setuid-root executables.
Vulnerable:
Not by default - but if the "nobody" user has uid 65535, yes.
Fixed in:
tar 1.11.8-8
Entered Distribution:
6 Feb 1997
