Security Information

Please send security-related bug reports to security@debian.org

Date reported: 08 May, 1998

Affected packages: super prior to 3.11.6

Brief description: files may be displayed without permissions

We have received a report that versions super were displaying files even if the particular user should not be able to read them. This has been forwarded to the upstream author, William Deich, who has released a fixed version.

If you have super installed on your machines we suggest that you upgrade immetiately. The problem is fixed in any 3.11.6 version of super.

After downloading the correct package, install using "dpkg -i (package_name).deb"

Vulnerable: Yes
