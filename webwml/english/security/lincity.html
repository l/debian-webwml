Date reported:
March 17, 1998
Affected packages:
lincity prior to version 1.09-3 with lincity-svga installed.
Brief description:
Potential buffer overruns.
User fix:
Check if lincity-svga is installed: dpkg -s lincity-svga
If it is: chmod a-s /usr/games/lincity (or wherever your executable lives).
This removes the setuid bits. Not a complete fix, but much safer.
