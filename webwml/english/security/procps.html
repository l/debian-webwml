Security Information

Please send security-related bug reports to security@debian.org

Date reported: 09 May, 1998

Affected packages: procps 1.2.6

Brief description:
We have received a report about procps 1.2.6 containing a file creation and corruption bug in XConsole. If you have procps installed on your machines we suggest that you upgrade immetiately. The problem is fixed in any 1.2.7 version of procps.

Vulnerable: Yes

Fixed in: Debian GNU/Linux 1.3.1 alias bo is not affected

Fixed in: Debian GNU/Linux pre2.0 alias hamm
