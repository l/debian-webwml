Security Information

Please send security-related bug reports to security@debian.org

Date reported: 30 May 1998

Affected packages: kdebase, various releases, see below

Brief description:
Shadow group vulnerable to buffer overflow exploits on systems running klock.
Also, kvt saves config as root, not user.

Vulnerable: Yes

Fixed in: Debian GNU/Linux 1.3.1 alias bo

There are no KDE packages for the current stable release of Debian GNU/Linux. However there are backward compiled packages located in bo-unstable. These packages contain vulnerable klock and kvt binaries.
