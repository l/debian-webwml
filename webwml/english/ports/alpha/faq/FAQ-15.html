15. How To Boot Linux/Alpha?

There are basically three ways of getting Linux/Alpha booted, depending on what firmware you're using. The preferred method of booting used to be to install MILO into the flash ROM. However, in that way newer hardware may not be initialized properly. If that is the case, one can boot MILO from within the ARC or AlphaBIOS console (Windows NT firmware).
Finally, if MILO has not been ported to a machine, it is still possible to boot Linux using the SRM firmware (DEC Unix). Installation of and booting with MILO is described in the the MILO Howto. Be sure to use a recent MILO! Booting with SRM is described in the SRM Howto.
