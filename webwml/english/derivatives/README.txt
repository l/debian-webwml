The goal of this file is to provide idea about how to match criteria
listed in derivatives page.
However the examples provided should be considered like inspiration and
not like hard rules: feel free to use others elements to check each
criteria.

 - actively cooperate with Debian:

Derivatives could file bugs to Debian BTS, forward patches,
sponsor DebConf, maintain packages in Debian.

 - actively maintained:

Package are updated regularly.
People responds to queries/bug reports.

 - notable distribution:

Notability can be shown by a documented Wikipedia page or being cited in
linux websites.

 - established distribution:

Derivative exists at least since 1 or 2 years, and must be on distrowatch.
